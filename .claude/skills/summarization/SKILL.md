# Summarization

Summaries should help the next decision.

Structure:

- One-line takeaway.
- Key points.
- Decisions or recommendations.
- Action items.
- Risks, gaps, or assumptions.
- Source references when available.

Do not flatten contradictions or overstate confidence.
