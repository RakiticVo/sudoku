# Writing

Write for the reader's next decision.

Before drafting:

- Identify audience, purpose, and desired action.
- Put the main point first.
- Use headings only where they help scanning.
- Separate facts, recommendations, and assumptions.
- Cut filler and unsupported claims.

For technical writing, include commands, file paths, and verification only when they are actionable.
