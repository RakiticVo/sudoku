# Web Search Strategy

Search deliberately.

Process:

- Translate the question into several query phrasings.
- Search for primary sources first.
- Use date filters when freshness matters.
- Compare independent sources for important claims.
- Track which searches were tried and what they found.

Do not rely on the first result when the decision matters.
