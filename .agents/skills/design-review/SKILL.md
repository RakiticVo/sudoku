# Design Review

Review the actual user experience.

Check:

- Visual hierarchy and scan path.
- Spacing, alignment, density, and rhythm.
- Color contrast and accessible states.
- Responsive behavior across key viewports.
- Component consistency and interaction feedback.
- Empty, loading, error, and long-content states.

Prefer screenshots or rendered verification when possible.
